Problem ID 55146

In an isosceles triangle, the sides are equal to either 3 or 7. Which side length is the base?

Hint

Use the triangle inequality.

Solution

Let a, a, b be the sides of the triangle. If a = 3, b = 7, then a + a = 2a $<$ b = 7, which is impossible.

Answer

The smaller one.
