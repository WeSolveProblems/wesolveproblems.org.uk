Problem ID 109946

Solve the equation {$($x + 1$)$^3} = $x^3$.

Solution

It follows from the definition of the fractional part that 0 $≤$ $x^3$ $<$ 1, that is, 0 $≤$ x $<$ 1. The equality {a + b} = a holds if and only if b is an integer. Therefore, the number n = $3x^2$ + 3x is an integer. The function y = $3x^3$ + 3x increases on [0, 1], y $($0$)$ = 0, y $($1$)$ = 6, therefore 0 $≤$ n $<$ 6, and from the roots of the equation $3x^2$ + 3x - n = 0 we choose the root lying on the interval [0, 1$)$.
