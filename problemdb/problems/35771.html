Problem ID 35771

Does a continuous function that takes every real value exactly 3 times exist?

Hint

An example can be found in the form of a sum of periodic and linear functions.

Solution

One of the possible options is a function whose graph is a "saw". Another option is the function f$(x)$ = 2x/3$\pi$ + sin x $. This is an example of a "smoothed saw".

Answer

Yes, such a function does exist.
