Problem ID 30423

Can there be exactly 100 roads in a state in which three roads leave each city?

Solution

If there are k cities in the state, then there are 3k/2 roads $($see problem number 87972a$)$. But such a number cannot be equal to 100.

Answer

There cannot be.
