Problem ID 88016

Decode this rebus: replace the asterisks with numbers such that the equalities in each row are true and such that each number in the bottom row is equal to the sum of the numbers in the column above it.

Solution

We shall replace all the asterisks in the rebus with different letters. Note that different letters could stand for the same number. Also note that the letter O is not used since it can cause confusion with the number zero. Thus the rebus takes the following form.

We can determine the value of some of the letters straight away. C = 0 (otherwise the result of the first row cannot end with 0); D = 1 (otherwise the result of the third column cannot begin with 1); A = B = 1 (if A>1 or B>1 then D$\neq$1); F = 1 (since we know A, B, C); L = N = 1; M = P = 0 (otherwise the equality in the third row cannot hold); Q = 1 (otherwise the equality in the fourth row cannot hold); T = 1 (otherwise the equality in the fifth row cannot hold).
Now we will write out the rebus again with these letters replaced by the corresponding numbers, as shown below.

Then: G = 7, 8 or 9 (otherwise the sum of the first column will be less than 100). In fact, G = 8 (since 67 and 69 are not divisible by any number ending with 7, while 68 is). Then H = 1, K = 4, U = 0, V = 1. If we rewrite the numbers WX and 1YZ from the fifth row as sums of the third and fourth columns, the fifth row becomes: 101 + (10 + 17 + 10 + R) = (110 + 4 + 20 + S), or S = R + 4. But S + R = 12 (from the fourth row). It follows that R = 4, S = 8. Then W = 4, X = 1, Y = 4, Z = 2. The final solution is below.
