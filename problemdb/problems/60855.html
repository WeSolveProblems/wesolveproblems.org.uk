Problem ID 60855

One of the roots of the equation $x^2 + ax + b = 0 is 1 + \sqrt 3$. Find a and b if you know that they are rational.

Solution 1

Substituting $x = 1 + \sqrt 3$ into the equation, we arrive at the equation $(4 + a + b) + (a + 2) \sqrt 3 = 0$. Since $\sqrt 3$ is an irrational number, this equality is possible only when 4 + a + b = 0 and a + 2 = 0.

Solution 2

The second root of this equation is the conjugate number $1 - \sqrt 3$ $($see the solution of problem number 60624$)$. Therefore,

$- a = (1 + \sqrt 3) + (1 - \sqrt3 ) = 2$, $b = (1 + \sqrt 3) (1 - \sqrt 3) = - 2$.

Answer

a = b = -2.
