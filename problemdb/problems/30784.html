Problem ID 30784

Prove that a graph, in which every two vertices are connected by exactly one simple path, is a tree.

Solution

Obviously, this graph is connected. Suppose now that there is a cycle in this graph. Then any two vertices of this cycle are connected by at least two simple paths. This is a contradiction.
