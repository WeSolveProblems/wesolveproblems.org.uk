Problem ID 61020

Construct the polynomial R $($x$)$ from the problem 61019 if:

a$)$ P $($x$)$ = $x^6$ - $6x^4$ - $4x^3$ + $9x^2$ + 12x + 4;

b$)$ P $($x$)$ = $x^5$ + $x^4$ - $2x^3$ - $2x^2$ + x + 1.

Solution

a$)$ P'$($x$)$ = $6x^5$ - $24x^3$ - $12x^2$ + 18x + 12 = 6 $($x^5 - 4x^3 - 2x^5 + 3x + 2$)$. Applying the Euclidean algorithm, we find that Q $($x$)$ = GCD $($P $($x$)$, P '$($x$)$)$ = $x^4$ + $x^3$ - $3x^2$ - 5x - 2, and R $($x$)$ = P $($x$)$ / Q $($x$)$ = $x^2$ - x-2 .

b$)$ The first method. P $($x$)$ = $5x^4$ + $4x^3$ - $6x^2$ - 4x + 1. Q $($x$)$ = GCD $($P $($x$)$, P '$($x$)$)$ = $x^3$ + $x^2$ - x-1, and R $($x$)$ = P $($x$)$ / Q $($x$)$ = $x^2$ - 1.

The second method. $x^5$ + $x^4$ - $2x^3$ - $2x^2$ + x + 1 = $($x^5 - 2x^3 + x$)$ + $($x^4 - 2x^2 + 1$)$ = $($x + 1$)$ $($x^2 - 1$)$^2$ = $($x + 1$)$^3 $($x - 1 $)$^2$. Consequently, R $($x$)$ = $($x + 1$)$ $($x-1$)$.

Answer

a$)$ $x^2$ - x - 2; b$)$ $x^2$ - 1.
