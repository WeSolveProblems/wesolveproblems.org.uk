Problem ID 109440

Solve the equation: $(x^3 - 2) (2^{sin x} - 1) + (2^{x^3} - 4) sin x = 0$.

Solution

From the fact that the function $y = 2^t$ increases, it follows that:

1) If sin x $>$ 0, then $2^{sin x} – 1 $ $>$ 0; if $sin x $<$ 0$, then $2^{sin x} - 1 <0$.

2) If $x^3 - 2> 0$, then $2^{x^3} - 4> 0$; if $x^3 – 2$ $<$ 0, then $2^{x^3} – 4$ $<$ 0.

Therefore, if $(x^3 - 2) (2^{sin x} - 1)$ $>$ 0$, then $(2^{x^3} - 4) sin x$>$ 0$; if $(x^3 - - 2) (2^{sin x} - 1) $<$ 0$, then $(2^{x^3} - 4) sin x $<$ 0$; that is, the signs of the expressions $(x^3 - 2) (2^{sin x} - 1)$ and $(2^{x^3} - 4) sin x$ coincide. Therefore, each term on the left hand side of the equation must vanish, that is, $x^3 = 2$ or sin x = 0.

Answer

$\sqrt [3] {2}, πn (n ∈ Z)$.
