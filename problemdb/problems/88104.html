Problem ID 88104

Is the number $10^{2002}$ + 8 divisible by 9?

Hint

Think of what numbers this number consists of.

Solution

The first digit of this number is 1, the last digit is 8, and between them the number 0 is repeated 2001 times. The sum of the digits is 9, so the number is divisible by 9.

Answer

It is divisible by 9.
