Problem ID 65480

Solve the equation 2 sin πx / 2 - 2 cos πx = $x^5$ + 10x - 54.

Solution

Transferring all the terms to one side, we obtain the equation $x^5$ + 10x - 54 - 2 sin πx / 2 + 2 cos πx = 0. Consider the function f $(x)$ = $x^5$ + 10x - 54 - 2 sin πx / 2 + 2 cos πx. Note that f $(x)$ = $5x^4$ + 10 - π cos πx / 2 - 2 sin πx $>$ $5x^4$ + 10 - 3π $>$ 0. Consequently, f $(x)$ is an increasing function, therefore the equation f $(x)$ = 0 has no more than one root. Since f $(2)$ = 32 + 20 - 54 - 2 sin π + 2 cos 2π = 0, then 2 is the root of the original equation.

Answer

2.
