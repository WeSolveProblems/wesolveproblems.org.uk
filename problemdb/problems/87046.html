Problem ID 87046

Let M be the point of intersection of the medians of the triangle ABC, and O an arbitrary point on a plane. Prove that

$OM^2$ = 1/3 $($$OA^2 + OB^2 + OC^2$$)$ - 1/9 $($$AB^2 + BC^2 + AC^2$$)$.

Hint

Let M be the intersection point of the medians of the triangle ABC. Then $\overline{OM}$ = 1/3 $($$\overline{OA}$ + $\overline{OB}$ + $\overline{OC}$$)$.

Solution

If M is the intersection point of the medians of the triangle ABC, then

$\overline{OM}$ = 1/3 $($$\overline{OA}$ + $\overline{OB}$ + $\overline{OC}$$)$,

so

$\overline{OM}^2$ = 1/9 $(\overline{OA} + \overline{OB} + $\overline{OC}$)^2$ = = 1/9 $($$OA^2 + OB^2 + OC^2$ + $2 \times $\overline{OA}$ \times $\ overline{OB}$$ + $2 \times $\overline{OA}$ \times $\overline{OC}$$ + $2 \times $\overline{OB}$ \times $\overline{OC}$$.

From the equality $\overline{AB}$ = $\overline{OB}$ - $ \overline{OA}$ it follows that $AB^2$ = $OB^2$ - 2 \times $\overline{OA}$ \times $\overline{OB}$ + $OA^2$, from which we find that

$2 \times \overline{OA}$ \times $\overline{OB}$ = $OB^2 + OA^2 - AB^2$.

Similarly, we find that

$\overline{BC}$ = $\overline{OC}$ - $\overline{OB}$, $BC^2$ = $OC^2$ - 2 \times $\overline{OB}$ \times $\ overline {OC}$ + $OB^2$, $\overline{AC}$ = $\overline{OA}$ - $\overline{OC}$, $AC^2$ = $OA^2$ - $2 \times $\overline{OA}$ \times $\overline{OC}$ + $OC^2$,

whence

$2 \times $\overline{OB}$ \times $overline{OC}$ = $OC^2 + OB^2 - BC^2$, 2 \times $\overline{OA}$ \times $\ overline{OC}$ = $OA^2 + OC^2 - AC^2$.

Consequently,

$\overline {OM}^{2}$ = 1/9 $($OA^2 + OB^2 + OC^2$ + $2 \times $\overline {OA}$ \times $\overline{OB}$ + $2 \times $\overline{OA}$ \times $\overline{OC} + $2 \times $\overline{OB}$ \times $\overline{OC}$)$ = 1/9 $($OA^2 + OB^2 + OC^2 + OB^2 + OA^2 - AB^2 + OC^2 + OB^2 - BC^2 + OA^2 + OC^2 - AC^2$)$ = = 1/9 $($3 \times OA^2$ + $3 \times OB^2$ + $3 \times OC^2$ - AB^2 - BC^2 - AC^2$)$ = = 1/3 $(OA^2 + OB^2 + OC^2) - 1/9 $(AB^2 + BC^2 + AC^2).
