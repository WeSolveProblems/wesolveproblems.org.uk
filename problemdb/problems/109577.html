Problem ID 109577

The function f $(x)$ is defined and satisfies the relationship $(x-1)$ f $((x=1)/(x-1))$ - f $(x)$ = x for all x $\neq$ 1. Find all such functions.

Solution

By substituting into this equation instead of x the fraction $\frac{x+1}{x-1}$, we get: $\frac{2}{x-1}$ $f (x) -f(\frac{x+1}{x-1})$ = $\frac{x+1}{x-1}$. $(1)$ Equation $(1)$, together with the equation given in the condition, defines a system of two equations with unknowns f $(x)$ and $f ( \frac{x+1}{x-1})$. Solving it, we find that f $(x)$ = 1 + 2x. Checking shows that the function found satisfies the equation.
