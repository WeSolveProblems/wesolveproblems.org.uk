Problem ID 60877

Prove that if $($m, 10$)$ = 1, then there is a repeated unit $E_n$ that is divisible by m. Will there be infinitely many repeated units?

Solution 1

By the Euler theorem, $ 9E_{kφ (m)} $ = $10^{kφ (m)}$ -1 $\equiv$ 0$ $($mod m$)$$ for any k $\geq$ 1. If m is not divisible by 3, then $ E_{kφ (m)} $ \equiv $ 0 $ ($mod m$) $. If m is divisible by 3 or by 9, then the number $E_{9kφ (m)} $ can be divided by m.

Answer

Yes, there will be.
