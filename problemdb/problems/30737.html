Problem ID 30737

How many ways can you build a closed line whose vertices are the vertices of a regular hexagon $($the line can be self-intersecting$)$?

Hint

The task is equivalent to counting the number of arrangements of six objects in a circle. See problem number 60373.

Solution

Nothing to translate.

Answer

5! = 120 ways.
