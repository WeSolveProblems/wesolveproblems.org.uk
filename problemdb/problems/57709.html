Problem ID 57709

Prove that the point X lies on the line AB if and only if $\overrightarrow{OX}$ = t $\overrightarrow{OA}$ + $($1 - t$)$ $\overrightarrow{OB}$ for some t and any point O.

Solution

The point X lies on the line AB if and only if $\overrightarrow{AX}$ = $\lambda$$\overrightarrow{AB}$, that is, $\overrightarrow{OX}$ = $\overrightarrow{OA}$ + $\overrightarrow{AX}$ = $(1 - \lambda) \overrightarrow{OA} + \lambda \overrightarrow{OB}.
