Problem ID 79595

a)$ Find f $(0)$ and f $(1)$. b$)$ Find all such functions f $(x)$.

Solution

Answer: There is only one such function f $(x)$, namely f $(x)$ = 2/$(1-2x)$ for x ≠ 1/2, f$(1/2)$ = 1/2 a$)$ We write down this equation for x = 0 and for x = 1: Solving this system of equations, we obtain f $(0)$ = 2, f $(1)$ = -2.

b$)$ We write this equation for x = ½ + t and for x= ½ - t For t ≠ 0, this system of equations has a unique solution f$(1/2 + t)$ = -1/t, f$(1/2 - t)$ = 1/t. Hence, for all such functions f for x ≠ ½, f $(x)$ = 2/$(1-2x)$. We now write this equation for x = 1/2. f $(1/2)$ + $(1/2 + 1/2)$ f $(1/2)$ = 1, that is f$(1/2)$ = 1/2 We can verify that this function is suitable by performing direct substitution.
