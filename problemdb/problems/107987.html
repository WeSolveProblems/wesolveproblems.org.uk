Problem ID 107987

To each pair of numbers x and y some number x * y is placed in correspondence. Find 1993 * 1935 if it is known that for any three numbers x, y, z, the following identities hold: x * x = 0 and x * $(y * z)$ = $(x * y)$ + z.

Solution

x = x * x + x = x * $(x * x)$ = x * 0 = x * $(y * y)$ = x * y + y. So, x * y = x - y. Therefore 1993 * 1935 = 1993 - 1935 = 58.

Answer

58.

Remarks

1. If we take x * y = x - y, then both identities are really satisfied.

2. Compare with problem no. 98169.
