Problem ID 65352

In a terrible thunderstorm, along a rope ladder, n dwarfs ascend in a chain. If suddenly there is a thunderbolt, then from fear, every gnome, regardless of others, can fall with probability p $($0 $<$ p $<$ 1$)$. If the dwarf falls, then he also hits all of the dwarfs that are below him. Find:

a) The probability that exactly k dwarfs will fall.

b) The mathematical expectation of the number of fallen dwarfs.

Solution

Let q = 1 - p be the probability that the gnome does not fall from the fright $($though, perhaps, he will fall, not from fear, but from the fact that someone else falls on top of him$)$.

a) Exactly k dwarfs will fall, only if the k-th gnome falls from fear, counting from below, and none of the n-k dwarfs that are above him will fall. Hence, the required probability is $pq^{n-k} = p (1 - p)^{n-k}$.

b) Let’s use indicators. Let the random variable $I_j$ be 1 if the j-th dwarf falls and 0 if he does not fall. The probability that the j-th dwarf does not fall is equal to the probability that he does not fall himself and that none of the j-1 dwarfs located above: $P (I_j = 0) = q^j$ fall. Therefore, $EI_j = P (I_j = 1) = 1 – q$.

Remarks

An event indicator is a random variable that is equal to one if the event occurred or is zero if the event did not occur. The mathematical expectation of the indicator is equal to the probability of this event.
