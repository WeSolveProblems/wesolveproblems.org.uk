Problem ID 21990

A 3x3 square is filled with the numbers $-1, 0, +1$ Prove that two of the 8 sums in all directions - each row, column, and diagonal - will be equal.

Solution

There are only 7 possible values for the sum - from $-3$ to $+3$.
