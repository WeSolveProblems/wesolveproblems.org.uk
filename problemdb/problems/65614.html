Problem ID 65614

Does there exist a function f $($x$)$ defined for all real numbers such that f $($sin x$)$ + f $($cos x$)$ = sin x?

Solution

Suppose that such a function exists. Then f $($sin 0$)$ + f $($cos 0$)$ = sin 0, that is, f $($0$)$ + f $($1$)$ = 0. But f $($sin π / 2$)$ + f $($cos π / 2$)$ = sin π / 2, that is, f $($0$)$ + f $($1$)$ = 1. This is a contradiction.

Answer

There does not exist.
