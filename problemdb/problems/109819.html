Problem ID 109819

Author: N. K. Agakhanov

Is there a bounded function f: $\mathbb{R} \rightarrow \mathbb{R}$ such that $f (1)> 0$ and $f (x)$ satisfies the inequality $f^2 (x + y) \geq f^2 (x) + 2f (xy) + f^2 (y)$ for all $x, y \in R$?

Solution

Let's take an arbitrary $x_1 \neq 0$ and set $y_1 = \frac{1}{x_1}$. Then $f^2 (x_1 + y_1) \geq f^2 (x_1) + 2f (1) + f^2 (y_1) \geq f^2 (x_1) + a$, where $a = 2f (1)> 0$. We will next choose $x_n = x_{n-1} + y_{n-1}, y_n = \frac{1}{x_2}$, for $n \geq 2$. Then $f^2 (x_n + y_n) f^2 (x_n) + a = f^2 (x_{n-1} + y_{n-1}) + a \geq f^2 (x_{n-1}) + 2a \geq ... \geq f^2 (x_1) + na$. It is clear that the sequence $f (x_1), f (x_2), f (x_n)$, is unbounded.

Answer

No, such a function does not exist.
