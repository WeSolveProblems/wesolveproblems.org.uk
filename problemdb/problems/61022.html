Problem ID 61022

For which A and B does the polynomial $Ax^{n + 1} + Bx^n + 1$ have the number x = 1 at least two times as its root?

Solution

$f (1) = A + B + 1, f '(1) = (n + 1) A + nB$. Solving the system: $A + B + 1 = (n + 1) A + nB = 0$, we find $A = n, B = - (n + 1)$.

Answer

$A = n, B = - (n + 1)$.
