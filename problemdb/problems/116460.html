Problem ID 116460

Pinocchio correctly solved a problem, but stained his notebook.

Under each blot lies the same number, which is not equal to zero. Find this number.

Solution

We replace the blots with the letter X: $($XX + XX + 1$)$ \times X $ = XXX. Having reduced by X, we obtain XX + XX + 1 = 111, whence XX = $($111 - 1$)$ / 2 = 55.

Answer

5.
