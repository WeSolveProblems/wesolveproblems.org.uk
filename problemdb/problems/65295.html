Problem ID 65295

A die is thrown six times. Find the mathematical expectation of the number of different faces the die lands on.

Solution

Let ξi be a random variable, equal to 1 if die falls with the face with i points facing upwards at least once and 0, if this does not happen even once. Then the number of different dropped faces is ξ = ξ1 + ... + ξ6. Moving on to the expectations, we get: Eξ = 6Eξ1, since all the quantities ξi are distributed identically.
