Problem ID 35733

A broken calculator carries out only one operation "asterisk": $a*b = 1 - a/b$. Prove that using this calculator it is possible to carry out all four arithmetic operations (addition, subtraction, multiplication, division).

Hint

Begin with division.

Solution

For division: $(a*b)*1=1-(a*b)/1=1-(1-a/b)=a/b$.
For multiplication: $a \times b = a/(1/b) = (a*((1*b)*1))*1$.
For subtraction: $a - b = (b*a) \times a = ((b*a)*((1*a)*1))*1$.
For addition: $a+b = a - (0-b) = a - ((b*0)*((1*0)*1))*1$.
