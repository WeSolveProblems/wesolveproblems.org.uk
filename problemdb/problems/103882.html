Problem ID 103882

Find the smallest four-digit number CEEM for which there exists a solution to the rebus MN + PORG = CEEM. $($The same letters correspond to the same numbers, different - different$)$.

Hint

C $>$ P, in particular, C $>$ 1.

Solution

Since C $>$ P, then C $>$ 1. Since we are looking for the smallest number, let's try to take P = 1, C = 2 and E = 0. Then $M \ geq 3$. The case CEEM = 2003 is possible: 35 + 1968 = 2003 or 38 + 1965 = 2003.
