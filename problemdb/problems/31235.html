Problem ID 31235

Find the last digit of the number $1 \times 2$ + $2 \times 3$ + ... + $999 \times 1000$.

Solution

$1 \times 2$ + $2 \times 3$ + ... + $999 \times 1000$ ≡ $1 \times 2$ + $2 \times 3$ + ... + $999 \times 1000$ + $1000 \times 1001$ ≡ 100 $($1 \times 2 + ... + $10 \times 11$)$ ≡ 0 $($mod 10$)$.

Answer

2.
