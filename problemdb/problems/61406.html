Problem ID 61406

Prove that if the function f $($x$)$ is convex upwards on the line [a, b], then for any distinct points $x_1, x_2$ in [a; b] and for any positive $\alpha_{1}$, $\alpha_{2}$ such that $\alpha_{1}$ + $\alpha_ {2}$ = 1 the following inequality holds:

f$(\alpha_1 x_1 + \alpha_2 x_2 ) > \alpha_1 f (x_1) + \alpha_2 f(x_2)$

Solution

Let l $($x$)$ be the tangent to the graph of the function f $($x$)$ at the point $\alpha_ {1}$ $x_1$ + $\alpha_{2}$ $x_2$. Then, by the definition of convexity,

$\alpha_1 f (x_1) + \alpha_2 f (x_2) < \alpha_1 l (x_1) + \alpha_2 l (x_2) = l (\alpha_1 x_1 + \alpha_2 x_2 ) = f (\alpha_1 x_1 + \alpha_2 x_2)$
