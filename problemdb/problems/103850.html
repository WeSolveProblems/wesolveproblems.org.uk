Problem ID 103850

The code of lock is a two-digit number. Ben forgot the code, but he remembers that the sum of the digits of this number, combined with their product, is equal to the number itself. Write all possible code options so that Ben could open the lock quickly.

Solution

Let the first digit of the code be x, and the second digit can be denoted as y. Then the number itself is written as 10x + y, and by the condition of the problem we can write the equation $($x + y$)$ + $x \times y$ = 10x + y. Therefore, $x \times y$ = 9x.

Since the code is a two-digit number, then x is not equal to 0, and therefore, y = 9. In this case, x can take any value except for 0. Check!

Answer

19, 29, 39, 49, 59, 69, 79, 89, 99.
