Problem ID 109438

The function f is such that for any positive x and y the equality f $($xy$)$ = f $($x$)$ + f $($y$)$ holds. Find f $($2007$)$ if f $($1/2007$)$ = 1.

Solution

For y = 1, this equation takes the form: f $($x$)$ = f $($x$)$ + f $($1$)$, hence f $($1$)$ = 0. Let x = 2007, y = 1/2007, then f $($1$)$ = f $($2007$)$ + f $($1/2007$)$, that is, f $($2007$)$ = - f $($1/2007$)$ = -1. Note that the simplest of functions satisfying the condition is a logarithmic function, which can be given by the formula f $($t$)$ = log 1 / 2007t. But this function is by no means the only possible one, since the condition for the continuity of the function f $($or its monotonicity$)$ is not given.

Answer

-1 .
