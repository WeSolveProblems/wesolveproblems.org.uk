Problem ID 65726

Author: A. Khrabrov

Do there exist integers a and b such that

a) the equation $x^2 + ax + b = 0$ does not have roots, and the equation $[x^2] + ax + b = 0$ does have roots? b) the equation $x^2 + 2ax + b = 0$ does not have roots, and the equation $[x^2] + 2ax + b = 0$ does have roots?

Note that here, square brackets represent integers and curly brackets represent non-integer values or 0.

Solution

a) The equation $x^2 - 3x + 3 = 0$ has no roots, since it has a discriminant negative, and the equation $[x^2] - 3x + 3 = 0$ has the roots $x_1 = 4/3, x_2 = 5/3$.

b) Suppose that the function $f (x) = x^2 + 2ax = b = (x + a)^2 + b - a^2$ does not vanish, i.e. it is equal to 0. Then $b - a^2$ is positive, and therefore at least 1 $($a and b are integers$)$. Answer

a) Yes, there do exist; b) No, there do not.
