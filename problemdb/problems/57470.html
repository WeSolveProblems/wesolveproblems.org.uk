Problem ID 57470

Prove that $\angle$ABC $<$ $\angle$BAC if and only if AC $<$ BC, that is, the larger side lies opposite the larger angle of the triangle, and opposite the larger side lies the larger angle.

Solution

It is sufficient to prove that if AC $<$ BC, then $\angle$ABC $<$ $\angle$BAC. Since AC $<$ BC, on the BC side, you can select $A_1$ so that $A_1C$ = AC. Then $\angle$ BAC $>$ $\angle$ $A_1AC$ = $\angle$ $AA_1C$ $>$ $\angle$ABC.
