Problem ID 102861

Rebus. Solve the numerical rebus AAAA-BBB + SS-K = 1234 $($different letters correspond to different numbers, but the same letters each time correspond to the same numbers$)$

Answer

2222 - 999 + 11 - 0 = 1234.
