Problem ID 109997

On a function f $(x)$ defined on the whole line of real numbers, it is known that for any a $>$ 1 the function f $(x)$ + f $(ax)$ is continuous on the whole line. Prove that f $(x)$ is also continuous on the whole line.

Solution

We use the following properties of continuous functions:

$(i)$ the sum and difference of continuous functions are continuous functions;

$(ii)$ if g $(x)$ is a continuous function, the function g $(ax)$ is also continuous.

Now note that the functions f $(x)$ + f $(2x)$ and f $(x)$ + f $(4x)$ are continuous by the condition, and with regards to $(ii)$, the function f $(x)$ + f $(2x)$, together with the function f $(2x)$ + f $(4x)$ is also continuous. Furthermore, by property $(i)$, the function $(f (x) + f (2x)) + (f (x) + f (4x)) - (f (2x) + f (4x))$ = 2f $(x)$, is continuous and, hence, so is the function f $(x)$.
