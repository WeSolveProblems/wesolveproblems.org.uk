Problem ID 35786

To transmit messages by telegraph, each letter of the alphabet is represented as a five-digit combination of zeros and ones corresponding to the binary number of the given letter in the alphabet $($letter numbering starts from one$)$. For example, the letter A is represented in the form 00001, letter B-00010, letter T-10100, letter Z-11010. Transmission of the five-digit combination is made via a cable containing five wires. Each bit is transmitted on a separate wire. When you receive a message, Cryptos has confused the wires, so instead of the transmitted word, a set of letters DWMYR is received. Find the word you sent.

hint

Note that as a result of the "entanglement" of the wires, the number of units in the representation of the transmitted letter does not change - this will greatly reduce the number of variants of the transmitted word.

Solution

Let's write out under each letter of the received text DWMYR all letters in which the representation has the same quantity of units, as the initial letter. Next, we try to "read" the meaningful text, choosing from each column of the resulting table one letter. The only option is the word HOUSE.
