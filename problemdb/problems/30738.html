Problem ID 30738

How many different four-digit numbers, divisible by 4, can be made up of the digits 1, 2, 3 and 4, a) if each number can occur only once?

b) if each number can occur several times?

Hint

Go through the options for the last two digits.

Answer

a) $2 \times 3$ = 6 numbers; b) $4^2 \times 4$ = 64 numbers.
