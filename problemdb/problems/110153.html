Problem ID 110153

The sum of the positive numbers a, b, c is π / 2. Prove that cos a + cos b + cos c $>$ sin a + sin b + sin c.

Solution

By the condition a + b $<$ π / 2, therefore cos a $>$ cos $(π / 2 - b)$ = sin b, since cos x decreases on the interval [0, π / 2]. Similarly, cos b $>$ sin c and cos c $>$ sin a. Adding these three inequalities, we obtain the required result.
