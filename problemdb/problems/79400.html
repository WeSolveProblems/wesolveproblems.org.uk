Problem ID 79400

We consider a function y = f $(x)$ defined on the whole set of real numbers and satisfying f $(x + k) \times (1 - f (x))$ = 1 + f $(x)$ for some number k â‰  0. Prove that f $(x)$ is a periodic function.

Solution

The condition implies that f $(x + k)$ = $(1 + f(x))$ / $(1 - f(x))$. Therefore f $(x + 2k)$ = - 1/f$(x)$; f $(x + 4k)$ = - 1/f$(x+2k)$ = f $(x)$. Hence f $(x)$ is a periodic function with a period of 4k.
