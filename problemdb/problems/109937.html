Problem

We are given a table of size $n \times n$. $n-1$ of the cells in the table contain the number $1$. The remainder contain the number $0$.

We are allowed to carry out the following operation on the table:

1. Pick a cell.

2. Subtract 1 from the number in that cell.

3. Add 1 to every other cell in the same row or column as the chosen cell.

Is it possible, using only this operation, to create a table in which all the cells contain the same number?

Solution

We will prove that any such initial table will contain a sub-table of size $2 \times 2$, in which there is one zero, and three ones.

There will always be a row in the table containing only 0s (as there are $n$ rows but only $n-1$ 1s). It follows that there will always be two adjacent rows, one of which contains only 0s, and the other contains at least one cell with the number 1.

This non-zero row will always contain two adjacent cells, one of which will contain the number 0 and the other the number 1. These two squares, along with the two adjacent squares in the row with only 0s make up the sub-table we are looking for.

Let us choose an arbitrary $2 \times 2$ sub-table. Let the value in the top left corner equal $a$, the top right corner $b$, the bottom left corner $c$ and the bottom right - $d$. After the operation is carried out on a random cell $X$, let these values be $a_1, b_1, c_1, d_1$ respectively.

Consider the expressions:

$$D = (a+d) - (b+c)$$ and $$D_1 = (a_1+d_1)-(b_1+c_1)$$.

Three scenarios are possible:

1) Cell $X$ lies outside of our sub-table, but lies in a row or column, which intersects with our sub-table. Without loss of generality, let $X$ lie in the same row as $a$ and $b$. Then $a_1=a+1$, $b_1=b+1$, $c_1=c$, $d_1=d$ and $D_1= (a+1+d)-(b+1+c)=D$.

2) Cell $X$ lies outside of our sub-table, and lies on neither a row, nor a column which intersects with our sub-table. Then, obviously $D_1=D$.

3) Cell $X$ lies within our sub-table. Without loss of generality, let it be the top left cell in the sub-table. Then $a_1=a-1$, $b_1=b+1$, $c_1=c+1$, $d_1=d$ and $D_1=D-3$. (If $X$ is the bottom right cell, then again $D_1=D-3$, otherwise $D_1=D+3$).

In this way, we have proved that the remainder when dividing $D$ by 3 is invariant. However in our initial table we found a $2 \times 2$ sub-table in which $D= \pm 1$, but we are tasked with creating a table in which all the numbers are the same - i.e. $D=0$, which is impossible.

Answer

No, it is not.
