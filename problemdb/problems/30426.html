Problem ID 30426

Is it possible to draw 9 segments on a plane so that each intersects exactly three others?

Solution

Consider a graph whose vertices are the given segments, and an edge connects two vertices when two corresponding segments intersect. It has 9 odd vertices, which contradicts problem number 87972 b).

Answer

It is not possible.
