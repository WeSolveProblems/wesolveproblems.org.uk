Problem ID 103858

Solve the rebus: $AH \times UX$ = 2001.

Solution

2001 = $3 \times 23 \times 29$. Therefore, the number 2001 can be represented as a product of two-digit numbers in the following ways: $69 \times 29$ or $23 \times 87$. Only the first option is suitable.

Answer

$29 \times 69$ = $69 \times 29$ = 2001.
