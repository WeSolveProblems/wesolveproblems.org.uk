Problem ID 66354

For all real x and y, the equality f $($x² + y$)$ = f $($x$)$ + f $($y²$)$ holds. Find f $($-1$)$.

Solution

Substituting x = 0, y = 0, we obtain f $($0$)$ = f $($0$)$ + f $($0$)$, that is, f $($0$)$ = 0.

Substituting x = 0, y = -1, we obtain f $($-1$)$ = f $($0$)$ + f $($1$)$, that is, f $($-1$)$ = f $($1$)$.

Substituting x = -1, y = -1, we obtain f $($0$)$ = f $($-1$)$ + f $($1$)$. This means that 2f $($-1$)$ = 0.
