78130 There is a system of equations

* x + * y + * z = 0,

* x + * y + * z = 0,

* x + * y + * z = 0.

Two people alternately enter a number instead of a star. Prove that the player that goes first can always ensure that the system has a non-zero solution.

Solution

Here is one possible strategy. The first player first writes an arbitrary coefficient in front of z in the first equation. Next, if the second player writes some coefficient in front of x or in front of y, then the first writes the same coefficient in the same equation for y or at x. If the second writes some coefficient at z, then the first writes an arbitrary coefficient for z in the remaining equation. The resulting system has a solution $(1, -1, 0)$.
