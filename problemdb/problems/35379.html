Problem ID 35379

Find all functions f $($x$)$ defined for all real values of x and satisfying the equation 2f $($x$)$ + f $($1 - x$)$ = $x^2$.

Hint

Substitute 1 - x instead of x.

Solution

In this equation, substitute 1 - x for x. We obtain the system

2f $($x$)$ + f $($1 - x$)$ = $x^2$,

f $($x$)$ + 2f $($1 - x$)$ = $($1 - x$)$², from which we find f $($x$)$ = 1/3 $($2x² - $($1 - x$)$ ²$)$ = 1/3 $($x² + 2x - 1$)$. Checking shows that the function found satisfies the condition.

Answer

f $($x$)$ = 1/3 $($x² + 2x - 1$)$.
