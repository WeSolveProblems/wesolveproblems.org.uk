Problem ID 35662

What weights can three weights have so that they can weigh any integer number of kilograms from 1 to 10 on weighing scales $($weights can be put on both cups$)$? Give an example.

Hint

Make the necessary search of variants.

Solution

The fact that this set really allows you to weigh any integer number of kilograms from 1 to 10 is shown by the following equations: 1 = 4-3, 2 = 9-3-4, 3 = 3, 4 = 4, 5 = 9-4, 6 = 9-3, 7 = 3 + 4, 8 = 3-4 + 9, 9 = 9, 10 = 4 + 9-3.

Answer

For example, weights of 3, 4, 9 kg.
