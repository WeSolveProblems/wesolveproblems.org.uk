Problem ID 61443

The exponent $y = e^x$ is a function for which the conditions $y '(x) = y (x)$ and $y (0) = 1$ are satisfied. Which sequence ${a_n}$ will have similar properties if the derivative is replaced by the difference operator $ \Delta $ ?

Answer

$a_n = 2^n$.
