Problem ID 60806

Without calculating the answer to $2^{30}$, prove that it contains at least two identical digits.

Hint

Apply the divisibility by 3 rule.

Solution

$2^{30}$ = $1024^3$ $>$ $10^9$. On the other hand, $1024^3$ < $2000^3$ = $8 \times $10^9$ $. Hence, the number $2^{30}$ is a ten-digit number. Next, see the solution to problem number 78742.
