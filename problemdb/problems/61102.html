Problem ID 61102

It is known that cos α$^{\circ}$ = 1/3. Is α a rational number?

Solution

Let α = m / n. Then cos $(360^{\circ} nα)$ = 1. Since cos $(360^{\circ} nα)$ = $T_{360n}$ $($ cos α $^{\circ}$)$ $($ for the definition of Chebyshev polynomials see problem number 61099 $)$, the polynomial $T_{360n}$ $(x)$ -1 tends to zero for x = cos $^{\circ}$ = 1/3. Therefore, the polynomial f $(x)$ = $2T_{360n}$ $(x / 2)$ -2 has a root of 2/3.
