Problem ID 65658

When 200 sweets are randomly distributed to a class of schoolchildren, there will always be at least two children who receive the same number of sweets or even no sweets at all. What is the minimum number of children in this class?

Solution

Assume that there are no more than 20 children in the class. Then we will give the first child in the class no sweets, the second - one sweet, the third - two sweets, and so on. In this case there will be no more than $0+1+2+...+19=190$ sweets handed out. If we give the remaining 10 sweets to the child with the most sweets, we will have demonstrated a distribution where no child has the same number of sweets as any other. Therefore there must be more than 20 children in the class.

If the class contains 21 children, then the minimum number of sweets necessary for each child to receive a different number of sweets is equal to $0+1+2+...+20=210$. Since we only have 200 sweets, then there will always be at least two children with the same number of sweets.

Answer

21.
