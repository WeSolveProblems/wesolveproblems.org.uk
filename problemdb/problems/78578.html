Problem ID 78578

All integers from 1 to 2n are written in a row. Then, to each number, the number of its place in the row is added, that is, to the first number 1 is added, to the second â€“ 2, and so on. Prove that among the sums obtained there are at least two that give the same remainder when divided by 2n.

Solution

Let the numbers $a_1, a_2, ..., a_{2n}$ be written in a row. Suppose that all the numbers $a_1$ + 1, $a_2$ + 2, ..., $a_{2n}$ + 2n give different remainders when divided by 2n. Then these remainders are equal to 1, 2, ..., 2n. Therefore, $($$a_1$ + 1$)$ + $($$a_2$ + 2$)$ + ... + $($$a_{2n}$ + 2n$)$ = 2$($1 + 2 + ... + 2n$)$ $\equiv$ 1 + 2 + ... + 2n $($mod 2n$)$. This is a contradiction, since 1 + 2 + ... + 2n = n$($2n + 1$)$ $\equiv$ n $($mod 2n$)$.
