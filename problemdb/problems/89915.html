Problem ID 89915

A bagel is cut into sectors. Ten cuts were made. How many pieces did this make?

Hint

Remember task 89914.

Solution

When a bagel is cut into pieces, the number of cuts and the number of sectors coincide, since one cut is needed in order to "make" a sector from the bagel.
