Problem ID 111805

Author: M.V. Murashkin

The sequence $(a_n)$ is given by the conditions $a_1 = 1000000, a_{n + 1} = n [a_n/n] + n$. Prove that an infinite subsequence can be found within it, which is an arithmetic progression.

Solution

We note that the numbers $k_n = a_{n+1}/n=[a_n n] +1$ are natural, and $k_{n + 1} = [a_{n + 1} n + 1] +1 = [nk_n/(n+1)] + 1 = k_n + [-k_n/(n+1)] + 1 \leq k_{n-1} + 1 = k_n$.

Hence, the sequence $k_n$ is non-increasing, and all of its terms are natural numbers. Then, starting from a certain point, it is constant, i.e. $k_n = k_{n + 1} = k_{n + 2} = .. = k$. This means that for $d \geq n$ we have that $a_{d + 2}-a_{d + 1} = (d + 1) k_d + 1-dk_d = (d + 1) k-dk = k$, that is, the whole subsequence $a_{n + 1}, a_{n + 2}, a_{n + 3}, ...$ is an arithmetic progression with difference k.
