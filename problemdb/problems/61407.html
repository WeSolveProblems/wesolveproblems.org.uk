Problem ID 61407

Inequality of Jensen. Prove that if the function f $(x)$ is convex upward on [a, b], then for any distinct points $x_1, x_2, ..., x_n (n \geq 2)$ from [a; b] and any positive $\alpha_{1}, \alpha_{2}, ..., \alpha_{n}$ such that $\alpha_ {1} + \alpha_{2} + ... + \alpha_{n}$ = 1, the following inequality holds: f $(\alpha_{1} x_1 + ... + \alpha_{n} x_n)$ $>$ $\alpha_{1}$ f $(x_1)$ + ... + $\alpha_{n}$ f $(x_n)$.

Solution

We use the method of induction. For n = 2, the Jensen inequality was proven in problem no. 61406. Suppose that it is true for some n $\geq$ 2 and subsequently prove it for n + 1. Let $\beta$ = $\alpha_ {1}$ + ... + $\alpha_{n}$ $>$ 0. Then $\alpha_1$ / $\beta_1$ + ... + $\alpha_n$ / $\beta_n$ = 1. Using the inequality with n = 2, we find f$(\alpha_1x_1 + ... + \alpha_{n+1}x_{n+1}) = f(\beta (\alpha_1 / \beta x_1 + ... + \alpha_n / \beta_1 x_n) + \alpha_{n+1}x_{n+1}) $>$ \alpha_{n+1} f(x_{n+1}) + \beta f(\alpha_1 / \beta x_1 + ... + \alpha_n / \beta x_n)$. Further, by the induction hypothesis, f $(\alpha_1 / \beta x_1 + ... + \alpha_n / \beta x_n) $>$ \alpha_1 / \beta f(x_1) + ... + \alpha_n / \beta f(x_n)$. Consequently, f $(\alpha_1 x_1 + ... + \alpha_{n+1} x_{n+1}) $>$ \alpha_1 f (x_1) + ... + \alpha_{n+1} f (x_{n+1})$.
