Problem ID 88105

Is the sum of the numbers 1 + 2 + 3 + ... + 1999 divisible by 1999?

Hint

Try to calculate the amount. Remember Carl Gauss.

Solution

Yes, it is, because 1 + 1998 = 2 + 1997 = ... = 999 + 1000 = 1999, i.e. this amount is equal to $1999 \times 999$ + 1999.

Answer

Yes.
