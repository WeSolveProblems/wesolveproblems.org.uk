Problem ID 65593

It is known that a $>$ 1. Is the equality = valid?

Solution

Since a $>$ 1, there is a positive integer n such that $n^4$ $≤$ a $<$ $($n + 1$)$^4$. Therefore, $n^2$ $≤$ <$($n + 1$)$^2$, and n $≤$ $<$ n + 1. Thus, = n. On the other hand, $n^2$ $≤$ $<$ $($n + 1$)$^2$, therefore n $≤$ $<$ n + 1, that is, = n. Hence, the above equality actually takes place.

Answer

Yes, it is valid.
