Problem ID 109707

Find all the functions f: $\mathbb {R} \rightarrow \mathbb {R}$ which satisfy the inequality f $(x + y)$ + f $(y + z)$ + f $(z + x)$ $\geq$ 3f $(x + 2y + 3z)$ for all x, y, z.

Solution

By substituting in x = y = -z, we obtain f $(2x)$ + f $(0)$ + f $(0)$ 3f $(0)$, whence f $(2x)$ $\geq$ f $(0)$. On the other hand, by substituting in x = z = -y, we obtain f $(0)$ + f $(0)$ + f $(2x)$ $\geq$ 3f $(2x)$, whence f $(2x)$ $\geq$ f $(0)$. Thus, f $(0)$ $\geq$ f $(2x)$ $\geq$ f $(0)$, that is, f $(2x)$ = C. It is easy to verify that for any C the function f $(x)$ = C satisfies the inequality.
