Problem ID 79504

Find the minimum for all α, β of the maximum of the function $y (x) = | cos x + α cos 2x + β cos 3x |$.

Solution

Answer: $\min \limits_ {\alpha, \beta} \max \limits_ {x} y (x) = {\frac {\sqrt3} {2}} $ and is achieved at α = 0, β = - $ {\frac {1} {6}} $. For all α and β, the following inequality is true $\max \limits_{x} y (x) ≥ \max (y (\frac {\pi} {6});$ $y ({\frac {5 \pi} {6}}))$ = $\max (| {\frac {\sqrt3} {2}} + {\frac {\alpha} {2}} |;$ $| -{\frac {\sqrt3} {2}} + {\frac {\alpha} {2}} |) ≥ ½ (| {\frac {\sqrt3} {2}} + {\frac {\alpha} {2}} | + | {-\frac {\sqrt3} {2}} + {\frac {\alpha} {2}}|) ≥ \frac {1} {2} |( {\frac {\sqrt3} {2}}$ + ${\frac {\alpha} {2}}) - (- \frac {\sqrt3} {2} + {\frac {\alpha} {2} } ) |$ = ${\frac {\sqrt3} {2}} $, so $ \min \limits _ {\alpha, \beta} \max \limits_ {x} y (x) ≥ {\frac {\sqrt3} {2}} $. We now put $f (x) = cos x - {\frac {1} {6}} cos 3x $ and find the extremum points of this function from the equation $f '(x) = 0: f' (x) = - sin x + {\frac {1} {2}} sin 3x = 0, -2 sin x + (3 sin x - 4 sin^3x) = 0$, from which either sin x = 0, x = kπ, or $sin^2x = {\frac {1} {4}}, x = ± {\frac {\pi} {6}} + kπ$. For the function $y (x) = | f (x) |$ we have $y (kπ) = {\frac {5} {6}} $<$ {\frac {\sqrt3} {2}}, y (± {\frac {\pi} {6}} + kπ) = {\frac {\sqrt3} {2}} $. It follows that $ \max \limits_{x} f (x) = {\frac {\sqrt3} {2}} $ and therefore $ \min \limits_{\alpha, \beta} \max \limits_ {x} y (x) ≤ {\frac {\sqrt3} {2}} $. $($This solution is from the book Halperin, Tolpygo$)$.
