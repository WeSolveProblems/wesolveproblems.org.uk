Problem ID 60668

Prove that if p is a prime number and 1 ≤ k ≤ p - 1, then $C_p^k$ is divisible by p.

Solution

If 0 $<$k $<$p, then $(k! (P - k) !, p) = 1$, therefore the number p in the numerator cannot be reduced.
