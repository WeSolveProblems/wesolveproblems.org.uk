Problem ID 88085

In one move, it is permitted to either double a number or to erase its last digit. Is it possible to get the number 14 from the number 458 in a few moves?

Hint

Try multiplying the original number by 2 until the first digit of the result is 7.

Solution

You can first double the number, then cross out the last digit, or you can do the reverse - first cross out the last digit, and then double the number. The value of the first digit of the result is almost not affected. Therefore, you can, for example, double the number until the first digit of the result is 7; cross out all of the digits except for the first; then double it. We get: 458, 916, 1832, 3664, 7328, 732, 73, 7, 14.

Answer

Yes: 458, 45, 90, 9, 18, 36, 72, 7, 14.
