Problem ID 111264

A continuous function f $(x)$ is such that for all real x the following inequality holds: f $(x^2)$ - $(f (x))^2$ $\geq$ 1/4. Is it true that the function f $(x)$ necessarily has an extreme point?

Solution

For x = 0 we obtain that f $(0)$ - $(f (0))^2$ $\geq$ ¼ $(f (0))^2$-f $(0)$ + ¼ $\leq$ 0 $(f (0) -1/2)^2$ 0 f $(0)$ =1/2. Similarly, for x = 1, f $(1)$ - $(f (1))^2$ $\geq$ ¼ f $(1)$ =1/2. Since the continuous function takes the same values at the ends [0; 1], then there is at least one maximum point or a minimum point inside this segment. Note that if x $\in$ [0; 1] f $(x)$ =1/2, then any point on this segment is an extremum point.

Answer

Yes, it is true.
