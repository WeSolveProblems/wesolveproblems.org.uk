Problem ID 88214

A class contains 38 pupils. Prove that within the class there will be at least 4 pupils born in the same month.

Solution

If in each month there were at most 3 pupils born, there would be as most $ 3 \times 12 = 36 $ students, but the problem states that there are actually 38.
