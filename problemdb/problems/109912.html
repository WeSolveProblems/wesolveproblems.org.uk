Problem ID 109912

For which α does there exist a function f:

$\mathbb{R} \rightarrow \mathbb{R}$ that is not a constant, such that f $(α (x + y))$ = f $(x)$ + f $(y)$ ;?

Solution

For α = 1 - one exists: f $(x)$ = x. For $α \neq 1$, for any x there exists a y such that y = α $(x + y)$: it is sufficient to set y =ax/$(1-a)$. But then from this equation we get f $(y)$ = f $(x)$ + f $(y)$, whence f $(x)$ = 0 for any x.

Answer

α = 1.
