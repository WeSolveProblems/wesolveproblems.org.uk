Problem ID 98427

In a row there are 1999 numbers. The first number is 1. It is known that each number, except the first and the last, is equal to the sum of two neighboring ones. Find the last number.

Solution

We denote our numbers by $a_1, a_2, ..., a_1999$. Adding the equalities $a_{n + 1}$ = $a_{n + 2}$ + $a_n$ and $a_{n + 2}$ = $a_{n + 3}$ + $a_{n + 1}$, we get $a_{n + 3}$ + $a_n$ = 0 or $a_{n + 3}$ = - $a_n$. Hence $a_{n + 6}$ = - $a_{n + 3}$ = $a_n$, that is, the sequence has a period of 6. Therefore, $a_1999$ = $a_{6 \times 333 + 1}$ = $a_1$ = 1.

Answer

1
