Problem ID 88003

There are five chain links with 3 rings in each. What is the smallest number of rings that need to be unhooked and hooked together to connect these links into one chain?

Hint

To connect two links, one ring is required.

Solution

We unhook 3 rings from one link. The remaining 4 links are connected by 3 unfastened rings.

Answer

3 rings.
