Problem ID 64891

The numerical function f is such that for any x and y the equality f $($x + y$)$ = f $($x$)$ + f $($y$)$ + 80xy holds. Find f $($1$)$ if f $($0,25$)$ = 2.

Solution

f $($0.5$)$ = f $($0.25 + 0.25$)$ = f $($0.25$)$ + f $($0.25$)$ + $80 \times 0.25 \times 0.25$ = 2 + 2 + 5 = 9. Similarly, f $($1$)$ = f $($0.5 + 0.5$)$ = f $($0.5$)$ + f $($0.5$)$ + $80 \times 0.5 \times 0.5$ = 9 + 9 + 20 = 38.

Answer

38.
