Problem ID 64411

Prove that for n$>$ 0 the polynomial $x^{2n + 1} - (2n + 1)x^{n + 1} + (2n + 1)x^n - 1$ is divisible by $(x - 1)^3$.

Solution

P $($1$)$ = 1 - $($2n + 1$)$ + $($2n + 1$)$ - 1 = 0,

P '$($1$)$ = $($2n + 1$)$ - $($n + 1$)$ $($2n + 1$)$ + n $($2n + 1$)$ = 0,

Pn $($1$)$ = 2n $($2n + 1$)$ -n $($n + 1$)$ $($2n + 1$)$ + n $($n-1$)$ $($2n + 1$)$ = n $($2n + 1$)$ $($2 - $($n + 1 $)$ + n - 1$)$ = 0.
