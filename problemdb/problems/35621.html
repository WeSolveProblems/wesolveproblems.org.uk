Problem ID 35621

Calculate $\int_0^{\pi/2}$ $(sin^2 (sin x) + cos^2 (cos x))$ dx.

Hint

Convert the integral using the reduction formula sin x = cos $(\pi/2 - x)$ and the basic trigonometric identity $sin^{2}x$ + $cos^{2}x$ = 1 .

Solution

Let's write: $\int_0^{\pi /2}$ $(sin^2 (sin x) + cos^2 (cos x))$ dx = $\int_0^{\pi/2}$ $(sin^2(sin x) + 1 - sin^2(sin (\pi/2 - x))$ dx. But $\int_0^{\pi/2}$ f $(x)$ dx = $\ int_0^{\pi/2}$ f$(\pi/2 - x)$ dx, since the graphs of the functions f $(x)$ and f $(\pi/2 - x)$ are symmetric with respect to the line x = $\pi/4 $, and the integrals of these functions on the interval from 0 to $\pi/2$ are equal. Thus, the required integral is equal to $\pi/2$.

Answer

$\pi/2$.
