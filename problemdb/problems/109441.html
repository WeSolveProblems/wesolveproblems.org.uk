Problem

We are given a 100x100 square grid and $N$ counters. All of the possible arrangements of the counters on the grid which follow the following rule are considered: no two counters lie in adjacent squares.

What is the largest value of $N$ for which in every single possible arrangement of counters following this rule, it is possible to find at least one counter, such that moving it to an adjacent square does not break the rule. (squares are considered adjacent if they share a side)

Solution

We will call an arrangement 'strict' if moving any counter to an adjacent square results in the rule being broken. A strict arrangement of 100 counters exists - place all the counters along the length of the longest diagonal.

That any arrangement of 99 (or fewer) counters is not strict, is effectively proved in problem 35395.

Answer

The largest value is $N=99$

Note

The question implies that all of the arrangements are considered for $N$ counters or fewer for each value of $N$
