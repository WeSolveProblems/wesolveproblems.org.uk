Problem ID 102797

Solve the equation f $(f (x))$ = f $(x)$ if

Solution

Note that f $(1)$ = 1 and f $(f (1))$ = 1, that is, x = 1 is the root of the given equation. The function f $(x)$ is decreasing, hence the function f $(f (x))$ is increasing. Consequently, the equation f $(f (x))$ = f $(x)$ has at most one root, that is, there are no other roots except x = 1.

Answer

x = 1.
