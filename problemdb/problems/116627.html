Problem ID 116627

Solve the inequality: $[x] \times {x}$ $<$ x - 1.

Solution

We denote: [x] = a, {x} = b, then x = a + b. This inequality takes the form: ab $<$ a + b - 1 ⇔ $($a - 1$)$ $($b - 1$)$ $<$0. Since 0 $≤$ {x} $<$ 1, then b $<$ 1. Then a $>$ 1, that is, [x] $>$ 1. Therefore, x $≥$ 2.

Answer

[2, + ∞].
