Problem ID 30787

In a graph, all the vertices have degree of 3. Prove that there is a cycle in it.

Solution

We consider an arbitrary connected component of this graph. It is not a tree, since it has no hanging vertex $($ see problem number 30786 $)$. Hence, there is a cycle in it.
