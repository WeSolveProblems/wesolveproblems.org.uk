Problem ID 64933

Find all of the solutions of the puzzle: ARKA + RKA + KA + A = 2014. (Different letters correspond to different numbers, and the same letters correspond to the same numbers.)

Solution

Since in the word ARKA there are A thousands, then A = 1 or 2. In addition, A + A + A + A ends in a 4, therefore A = 1. Since K + K + K ends in a 1, then K = 7. Since R + R + 2 = 10, then R = 4.

Answer

1471 + 471 + 71 + 1 = 2014.
