Problem ID 110035

Author: E. Znak

Does there exist a function f $($x$)$ defined for all $x \in \mathbb{R}$ and for all $x, y \in \mathbb{R}$ satisfying the inequality $| f (x + y) + sin x + sin y | < 2$?

Solution

Suppose that such a function f $($x$)$ exists. Putting $x = \pi /2, y = \pi /2$, we obtain $| f (π) +2 | $<$2$, i.e. $f (π) $<$0$. At the same time, for $x = 3\pi /2 , y = - \pi /2$ - we obtain $| f (π) -2 | $<$2$, i.e. $f (π)> 0$. This is a contradiction.

Answer

Such a function does not exist.
