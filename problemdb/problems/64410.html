Problem ID 64410

Prove that for n $>$ 0 the polynomial P $($x$)$ = $n^2$x^{n + 2}$ - $($2n^2 + 2n - 1$)$ $x^{n + 1}$ + $($n + 1$)$^2$x^n$ - x - 1 is divisible by $($x - 1$)$^3$.

Solution

P $($1$)$ = $n^2$ - $($2n^2 + 2n-1$)$ + $($n + 1$)$^2$ - 2 = 0,

P '$($1$)$ = $($n + 2$)$n^2$ - $($n + 1$)$ $($2n^2 + 2n-1$)$ + n $($n + 1$)$^2 - 1 = $n^2$ - $($n + 1$)$ $($n^2 + 2n + 1$)$ + 2 $($n + 1$)$ + n $($n + 1$)$^2$ - 1 = $n^2$ - $($n + 1$)$^2$ + 2n + 1 = 0,

P"$($1$)$ = $($n + 2$)$ $($n + 1$)$ $n^2$ - n $($n + 1$)$ $($2n^2 + 2n-1$)$ + n $($n-1$)$ $($n + 1$)$^2$ = n $($n + 1 $)$ $($($n + 2$)$n - $($2n^2 + 2n-1$)$ + n^2 - 1$)$ = 0. This means that P $($x$)$ is divisible by $($x - 1$)$^3$.
