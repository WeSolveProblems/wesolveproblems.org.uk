Problem ID 61001

Prove the following formulae are true:

$a^{n + 1}$ – $b^{n + 1}$ = $($a - b$)$ $($$a^n$ + $a^{n-1}b$ + ... + $b^n$$)$;

$a^{2n + 1}$ + $b^{2n + 1}$ = $($a + b$)$ $($$a^{2n}$ – $a^{2n-1}b$ + $a^{2n-2}$$b^2$ - ... + $b^{2n}$$)$.

Hint

Expand the brackets or find the quotients using Horner's scheme.
